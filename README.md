# Student Grade Tracker

A Java Swing-based application developed as part of the CodeAlpha Java Programming Internship.

## Features

* Add Student Records
* Grade Classification (A, B, C, D, F)
* Calculate Average Score
* Find Highest Score
* Find Lowest Score
* User-Friendly GUI using Java Swing

## Technologies Used

* Java
* Java Swing
* ArrayList
* Object-Oriented Programming (OOP)
